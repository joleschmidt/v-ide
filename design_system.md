# Design System: VØIDE (Updated)

## 1. Brand Identity

- **Name**: VØIDE

- **Slogan**: "Step into the VØIDE."

- **Vibe**: Nordic, Wilderness, Survival, Minimal, Natural.

## 2. Color Palette ("Night Forest")

### Backgrounds

- **Radar Black**: `#121212` (Main Background - NOT pure black, but very close)

- **Charcoal**: `#1a1a1a` (Cards / Containers)

- **Grid Lines**: `#2a2a2a` (Subtle dividers)

### Accents

- **Forest Green**: `#4a7a4a` (Primary Action / Selected State / Success)

- Use for "Buchen", CTAs, verified badges, and interactive elements.

- **Moss Green**: `#5a8a5a` (Hover States / Secondary Actions)

- **Pine Green**: `#3a5a3a` (Muted Accents / Icons)

### Text

- **Bone White**: `#e2e8f0` (Headlines - avoid pure `#ffffff` to reduce eye strain in dark)

- **Light Gray**: `#a8a8a8` (Body Text)

- **Muted Gray**: `#666666` (Disclaimers / Footer / Secondary Info)

## 3. Typography

### Headlines

- **Font**: Inter (Bold/Semi Bold)

- **Style**: Clean, readable, slight letter-spacing.

- **Example**: `Sektor Alpha`

### Data & Technical Text

- **Font**: JetBrains Mono, Roboto Mono, or system monospace.

- **Use for**: Coordinates, Dates, Prices, Technical Details.

- **Style**: Small, precise, clean.

## 4. UI Components

### The "Sector Card"

- Subtle rounded corners (4px-6px) for a modern, refined look.

- Image has natural, organic presentation.

- Info overlay uses dark background with subtle transparency.

### Icons

- Use outlined icons (Lucide React).

- Stroke width: 1.5px or 2px.

- Style: Clean, minimal, not overly aggressive.

### Buttons

- **Primary**: Solid Forest Green (`#4a7a4a`), clean text.

- **Secondary**: Transparent with Gray Border (Ghost Button).

- **Hover**: Moss Green (`#5a8a5a`) for primary actions.

## 5. Design Principles

- **Natural over Tactical**: Embrace the wilderness aesthetic without military aggression.

- **Minimal**: Let content breathe, use whitespace generously.

- **Greyscale Foundation**: Use greens sparingly as thoughtful accents.

- **Readability First**: Ensure sufficient contrast for all text elements.
